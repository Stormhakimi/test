
石油公司司机路线调度工具 

## 功能

- 📍 **地点库管理** — 维护常用地点（油井、仓库、加油站等）
- 🚗 **司机管理** — 管理司机和车辆信息
- 📋 **智能调度** — 选择地点和司机，自动计算最短路线
- 🗺️ **地图可视化** — 在高德地图上查看路线
- 📱 **微信通知** — 一键生成路线文本，复制到微信发给司机
- 📅 **历史记录** — 查看过去的调度计划
- 💾 **数据备份** — 导出/导入 JSON 备份文件

## 快速开始

### 1. 准备高德地图 API Key（必须）

地图功能需要高德地图 API Key：

1. 访问 [console.amap.com](https://console.amap.com) 注册账号
2. 用支付宝完成实名认证
3. 创建新应用，选择 **Web端(JS API)**
4. 复制生成的 Key

### 2. 部署到 Vercel（推荐，免费）

1. Fork 或克隆此项目到 GitHub
2. 访问 [vercel.com](https://vercel.com) 并用 GitHub 账号登录
3. 点击 "New Project"，选择你的 GitHub 仓库
4. Vercel 会自动检测 Vite 项目并部署
5. 部署完成后，你会获得一个 URL（如 `https://油田调度.vercel.app`）

### 3. 首次使用

1. 打开应用 URL（手机浏览器也可）
2. 点击底部 **"设置"** 标签
3. 在 "高德地图 API Key" 处输入你的 Key，点击保存
4. 在 "出发地" 处填写公司/车场地址，点击保存
5. 前往 **"地点"** 标签，添加常用地点
6. 前往 **"司机"** 标签，添加司机信息
7. 点击 **"调度"** 标签开始规划路线！

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 打包构建
npm run build
```

## 注意事项

- **数据存储**：所有数据保存在浏览器本地，清除浏览器数据会丢失记录
- **建议定期备份**：在"设置"→"数据管理"中导出 JSON 备份文件
- **高德 API 免费额度**：Web JS API 每月 150,000 次调用，日常使用完全够用
- **隐私安全**：数据不会上传至服务器，仅在你的设备上存储
